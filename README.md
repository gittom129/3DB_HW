This is the assignment to upload the data from .csv file to the database and export the data from database back to download folder in .csv files
the home page has buttons for you to choose the  files and click upload to the corresponding table. duplicated records will be rejected
when export button is clicked, it will download to your download folder
a clear button at the bottom help to clear all the data in the database.
There are tables for customers, Books, Bookshops and BookAvailability
Sample data are provided in the csv_file folder.
